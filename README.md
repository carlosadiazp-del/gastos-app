## 🚀 Publicar en GitHub Pages (GRATIS)

### Paso 1 - Crear cuenta en GitHub
Ve a https://github.com y crea una cuenta gratis con tu email.

### Paso 2 - Crear repositorio
1. Clic en el botón verde **"New"**
2. Nombre del repositorio: `gastos-app`
3. Selecciona **"Public"**
4. Clic en **"Create repository"**

### Paso 3 - Subir archivos
1. Clic en **"uploading an existing file"**
2. Arrastra y suelta TODOS estos archivos:
   - `index.html` (el GastosApp.html renombrado)
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Clic en **"Commit changes"**

### Paso 4 - Activar GitHub Pages
1. Ve a **Settings** del repositorio
2. Clic en **"Pages"** (menú izquierdo)
3. En "Source" selecciona **"main"** branch
4. Clic en **"Save"**
5. Espera 2-3 minutos

### Paso 5 - Tu URL lista
Tu app estará disponible en:
`https://TU_USUARIO.github.io/gastos-app/`

## 📱 Cómo instalar en el celular

### Android (Chrome):
1. Abre la URL en Chrome
2. Aparecerá un banner: **"Agregar a pantalla de inicio"**
3. O toca el botón **📲 Instalar App** dentro de la app
4. Confirma la instalación

### iPhone (Safari):
1. Abre la URL en Safari
2. Toca el botón de compartir 📤
3. Selecciona **"Añadir a pantalla de inicio"**
4. Confirma
